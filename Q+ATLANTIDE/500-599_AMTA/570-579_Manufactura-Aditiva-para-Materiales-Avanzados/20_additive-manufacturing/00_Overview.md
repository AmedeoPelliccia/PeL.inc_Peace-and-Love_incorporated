# AMTA 570-579 · Section 07 · Subsection 20 — additive manufacturing

## 1. Purpose

Overview entry-point for the *additive manufacturing* subsection within the `570-579` code range (Section `07` — *Manufactura Aditiva para Materiales Avanzados*) of the **AMTA** architecture band (*Advanced Material, Bio & Nanotechnology Architecture*, master range `500–599`).

This subsubject (`00 Overview`) introduces the AMTA 570-579.20.00 slice and links it to the controlled Q+ATLANTIDE baseline[^baseline] and to the applicable industry standards listed in §4.

## 2. Scope

- Covers the *additive manufacturing* slice of the parent code range `570-579`.
- Inherits Q-Division authority and ORB support from the parent row in [`../../README.md` §3](../../README.md#3-architecture-table)[^archtable].
- Subsequent subsubjects (`01`–`99`) under this subsection extend this Overview with detailed data modules per S1000D[^s1000d].
