# ACV 790-799 · Section 09 · Subsection 010 — Operators

## 1. Purpose

Overview entry-point for the *Operators* subsection within the `790-799` code range (Section `09` — *Modelos de Negocio y Ecosistemas UAM*) of the **ACV** architecture band (*Aerial City Viability / UAM Architecture*, master range `700–799`).

This subsubject (`00 Overview`) introduces the ACV 790-799.010.00 slice and links it to the controlled Q+ATLANTIDE baseline[^baseline] and to the applicable industry standards listed in §4.

## 2. Scope

- Covers the *Operators* slice of the parent code range `790-799`.
- Inherits Q-Division authority and ORB support from the parent row in [`../../README.md` §3](../../README.md#3-architecture-table)[^archtable].
- Subsequent subsubjects (`01`–`99`) under this subsection extend this Overview with detailed data modules per S1000D[^s1000d].
