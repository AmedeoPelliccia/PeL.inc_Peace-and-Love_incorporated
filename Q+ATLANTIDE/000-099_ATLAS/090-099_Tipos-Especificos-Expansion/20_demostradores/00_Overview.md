# ATLAS 090-099 · Section 09 · Subsection 20 — demostradores

## 1. Purpose

Overview entry-point for the *demostradores* subsection within the `090-099` code range (Section `09` — *Tipos Específicos & Expansión*) of the **ATLAS** architecture band (*Aircraft Top-Level Architecture System*, master range `000–099`).

This subsubject (`00 Overview`) introduces the ATLAS 090-099.20.00 slice and links it to the controlled Q+ATLANTIDE baseline[^baseline] and to the applicable industry standards listed in §4.

## 2. Scope

- Covers the *demostradores* slice of the parent code range `090-099`.
- Inherits Q-Division authority and ORB support from the parent row in [`../../README.md` §3](../../README.md#3-architecture-table)[^archtable].
- Subsequent subsubjects (`01`–`99`) under this subsection extend this Overview with detailed data modules per S1000D[^s1000d].
